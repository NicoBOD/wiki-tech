# Wiki Tech

Wiki technique personnel de NicoBOD — notes, découvertes, débugs et astuces.

Thèmes : systèmes, réseaux, cloud, cybersécurité, IA, automatisation, logiciels.

**Consulter le wiki :** [nicobod.github.io/wiki-tech](https://nicobod.github.io/wiki-tech)

## Ajouter du contenu

1. Cloner le repo : `git clone git@github.com:NicoBOD/wiki-tech.git`
2. Ajouter un fichier `.md` dans le dossier `docs/<catégorie>/`
3. Commit et push sur `main` — le site se met à jour automatiquement
